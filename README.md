# CALCULATOR

Simple web-based calculator demo (HTML, CSS, JavaScript).

## Description
A minimal calculator UI implemented with `index.html`, `style.css`, and `script.js`. Click buttons or type on your keyboard to perform basic arithmetic.

## Features
- Addition, subtraction, multiplication, division
- Keyboard input support
- Responsive, single-file frontend (no build step)

## Run locally
1. Open `index.html` in your browser.
2. (Optional) Use VS Code Live Server: right-click `index.html` → _Open with Live Server_.

## Files
- `index.html` — UI
- `style.css` — styling
- `script.js` — calculator logic
